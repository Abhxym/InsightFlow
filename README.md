# InsightFlow

InsightFlow is a full-stack, enterprise-grade Survey Analytics Platform inspired by tools like Qualtrics and Forsta HX. It enables organizations to create dynamic surveys, collect and validate responses, analyze results through interactive dashboards, and generate AI-powered insights from open-ended feedback.

## Overview
The platform is designed to replicate the complete survey lifecycle—from survey creation to business reporting—while showcasing modern software engineering practices, business intelligence concepts, and data analytics techniques.
