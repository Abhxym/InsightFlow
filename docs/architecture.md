# Architecture

This project is divided into two primary subsystems: a Node.js API backend and a React frontend.

## Backend
- **Environment**: Node.js
- **Framework**: Express.js
- **Database**: MySQL
- **Authentication**: JWT (JSON Web Tokens)
- **Security**: bcrypt (password hashing), cors

## Frontend
- **Environment**: React (via Vite)
- **Routing**: React Router
- **HTTP Client**: Axios
- **Styling**: Vanilla CSS (Rich modern aesthetics)
